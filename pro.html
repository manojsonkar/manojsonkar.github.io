More About Project


A network service is a service hosted on a computer network. Network services provide some functionality for members or users of the network. Services are usually based on a defined service protocol. Network services are hosted by servers to provide Network services are configured on corporate local area networks to ensure security and user friendly operation. They help the LAN run smoothly and efficiently. Corporate LANs use network services such as DNS (Domain Name System) to give names to IP and MAC addresses (people remember names like “nm.lan” better than they remember numbers like “210.121.67.18”), and DHCP to ensure that everyone on the network has a valid IP address.
