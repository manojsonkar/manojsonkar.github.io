project title: Snake and Ladder Game

SKILLS UGE: C and graphics

DESCRIPTION

It is played between two or more players on a gameboard having numbered, gridded squares.
A number of "ladders" and "snakes" are pictured on the board, each connecting two specific board squares. there is a random number generator which implement with the help of RAND() function. according to random number dice of player will automatically increment by that numbers then program will check 2 condition.
1 is new number after increment come at mouth of snake? if yes then it will go to tail of snake. else if no then there will be no change.
2 is new number after increment come at beginning of ladder? if yes then it will go to upper most position of ladder through he ladder. else if no then there will be no change.
